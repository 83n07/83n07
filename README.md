## Current Work

**AMS-Calib** — A full-stack ROS2/Qt6 application for automated 
multi-sensor calibration of industrial robot arms, developed as part 
of a PhD research project at THWS. The system handles intrinsic, 
extrinsic, multi-camera, hand-eye, and camera-IMU calibration from 
a single interface, with autonomous robot motion generation and 
embedded RViz visualisation.

Project page: [robotik.thws.de/forschung/projekte/ams-calib](https://robotik.thws.de/forschung/projekte/ams-calib)

## Bachelor Thesis — Lufthansa Technik AG

**6D Pose Estimation of Unseen Aircraft Components using 3D Scan 
Data and RGB-D Images**

Implemented a fixtureless 6D pose estimation pipeline for aircraft 
hydraulic components using FoundationPose and SAM-6D. The system 
requires only a 3D scan, RGB image, depth image, and camera 
intrinsics — no manual annotation — and achieved approximately 
0.2mm translational accuracy post-calibration.

Conducted at the iMRO Service Centre, Lufthansa Technik AG, Hamburg, 
in collaboration with the Institute for Robotics (CERI), THWS.
